# thenowdesigns

Static starter site for showcasing visuals, art, design, and website add-ons.

## Files

- `/tmp/workspace/hmatrades/thenowdesigns/index.html` - main showcase page
- `/tmp/workspace/hmatrades/thenowdesigns/styles.css` - shared site styling

## Local preview

Open `index.html` directly in a browser, or serve the repository with any static file server such as:

```bash
cd /tmp/workspace/hmatrades/thenowdesigns
python3 -m http.server 8000
```
